Welcome to Image Gallery

Your one-stop solution for browsing and viewing beautiful images

Introduction

Image Gallery is a simple web application that allows you to browse through a collection of images provided by an external API. You can view author names and their images in a list format, and explore a gallery of images.

Key Features

  • Responsive and visually appealing UI
  • Author names and image thumbnails in an ag-grid
  • Image gallery with lazy loading
  • Search functionality for filtering data in the ag-grid
  • Pagination support in the ag-grid for navigating through different pages of images

Getting Started

To begin exploring the images, use the navigation tabs at the top of the page:

  • Author Image Listing: View a list of authors and their images in a tabular format.
  • Image Gallery: Show gallery of images.
